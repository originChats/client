# Contributor Covenant Code of Conduct

Everyone interacting in the OriginChats project's codebases, issue trackers, chat rooms and mailing lists is expected to follow this code of conduct.

The short version:
- Be respectful and kind.
- Assume good faith.
- Do not harass or discriminate against others.

If you experience or witness unacceptable behavior, contact the project maintainers (repository owner) with details. Contributors are expected to follow these rules; maintainers will enforce them when necessary.

For the full policy, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
